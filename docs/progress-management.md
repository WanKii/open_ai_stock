## 11. 验收门禁
### M1 门禁
- 设置页可保存并读取本机配置文件
- OpenAI、Anthropic、Tushare、AKShare、BaoStock 连接测试入口可用

### M2 门禁
- 基本信息、历史数据、财务数据、新闻数据同步任务可手动触发
- DuckDB 中存在标准化后的目标表

### M3 门禁
- 分析任务可排队、执行、保存 Agent 结果和总结结果
- 单角色失败不会导致整个任务必然失败

### M4 门禁
- 历史记录页可筛选、查看详情
- 日志页可按任务 ID 追踪

### M5 门禁
- 核心流程端到端测试通过
- 已知高优先级问题关闭
- MVP 发布说明可用

## 12. 当前推进建议
- 先做 M0 和 M1，保证配置、路由、页面框架和接口骨架先稳定
- 再做 M2 和 M3，优先打通“同步 + 分析”主链路
- 最后收口到 M4 和 M5，避免过早投入报表细节

截至 2026-04-09，项目处于“计划已明确、尚未开工”状态。建议以里程碑驱动推进，所有任务统一进入本文件维护，避免需求、进度、问题分散在多个地方。
